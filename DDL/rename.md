# RENAME문

### 실습용 테이블 생성

```
CREATE TABLE RENAME_TEST
(
    no number(10)
);
```

## 1. 테이블 이름 변경

### 사용문법

```
RENAME 기존테이블명 TO 바꿀테이블명
```

### 사용예제

```
RENAME RENAME_TEST TO RENAME_TABLE_TEST;
```
