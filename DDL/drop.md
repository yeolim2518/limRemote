# DROP문

### 실습용 테이블 생성

```
CREATE TABLE DROP_TEST
(
    no number(10)
);
```

![image](https://user-images.githubusercontent.com/42727909/49848972-f3f00b80-fe1a-11e8-8587-dccc0e379cc6.png)

## 1. 테이블 삭제

### 사용문법

```
DROP TABLE 테이블명
```

### 사용예제
```
DROP TABLE DROP_TEST;
```

위의 코드를 실행하면 테이블이 삭제된 것을 확인할수 있습니다.
